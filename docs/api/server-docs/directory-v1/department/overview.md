# Department Management Overview

Department Management Interface Overview

## Methods

**Create department**

    For creating departments under an enterprise

**Update department**

    Used to update department information. Update only the part of the explicitly passed parameter

**Delete department**

    For deleting departments.

**Get departmental information in bulk**

    It is used to query the details of the department according to the ID of the department in batches.

**Get the department list in bulk**

    It is used to obtain a list of eligible department details in batches according to the specified conditions.

**Search department**

    For searching for department information
