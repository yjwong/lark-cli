# Group disbanded

This event is triggered when a group is disbanded.{Usage Examples}(url=/api/tools/api_explore/api_explore_config?project=im&version=v1&resource=chat&event=disbanded)

> Notes:
> - The bot ability needs to be enabled.
> - You need to subscribe to Group disbanded events under the IM category.
> - The event is pushed to bots who have subscribed to this event in the group.
> - Disbanding the group and checking [Keep chat history for group members] will not trigger this event.
