# Create a group

Creates a group and sets the profile photo, name, and description for the group.

> Notes:
> - The bot ability needs to be enabled for the app.
> - This API is only used for creating a group. If you need to add users or bots into a group, refer to the Adds users or bots into a group chat.
> - Store apps cannot use user_id. Therefore, when using the store application to call this interface, please select open_id or union_id as the user ID type.
