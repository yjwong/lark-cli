# Group ID description
## What is chat_id？
`chat_id` is the ID that uniquely identifies a group chat. It is automatically generated by the system when a group chat is created.

`chat_id` is a string that starts with `oc_`, for example, `oc_a0553eda9014c201e6969b478895c230`.

## How to obtain the group ID？
### Scenario 1: An existing group

1. Create an app and enable the bot ability;
2. Add the bot into the group;
3. Obtain `chat_id` by calling the Obtain the list of groups with the user or bot API;
4. Use the `chat_id` as a parameter to Send messages or Add users or bots to a group chat.   

### Scenario 2: A new group created by the bot 
1. Create an app and enable the bot ability;
2. Create a group as a bot, and call the Create a group API to obtain `chat_id` from the response result;
3. Use the `chat_id` as a parameter to Send messages or Add users or bots to a group chat.
