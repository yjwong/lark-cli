# Obtain custom user properties
> In order to improve the security level, we have upgraded it. Please migrate to the new version as soon as possible. New version >>

This API is used to obtain the custom user properties set by the organization.

::: note
1. Before invoking the API, first confirm if the organization administrator has enabled "Allow to invoke by Open Platform's Contacts API" under "[Admin Console" > "Organization">"Member Profile](http://www.feishu.cn/admin/contacts/employee-field-new/custom)"
2. Apps invoking this API must have permission to read or update the current company’s Contacts.

## Request
| Facts |  |
| --- | --- |
| HTTP URL | https://open.larksuite.com/open-apis/contact/v1/tenant/custom_attr/get |
| HTTP Method | GET |
| Required scopes Enable any scope from the list | Access department's organize information Read contacts | ### Request header
| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| Authorization | string | Yes | `tenant_access_token` **Value Format**: "Bearer `access_token`" **Example Value**: "Bearer t-7f1bcd13fc57d46bac21793a18e560" Learn more about obtaining and using access_token. |
| Content-Type | string | Yes | **Fixed value**: "application/json; charset=utf-8" | ## Response
### Response body

|Parameter|Description|
|-|-|
|code|Error code, anything but 0 indicates failure|
|msg|Response code description|
|data|Response business information|
|  ∟is_open|If organization open custom user properties|
|  ∟custom_attrs|When is_open is true, this field will be returned. The field contains the custom user properties set by the organization. Each property includes a custom property ID generated by the platform and localized property names set by the organization, zh_cn, en_us, ja_jp represent the Chinese, English and Japanese names
### Response body example
```json
{
    "code": 0,
    "msg": "success",
    "data": {
        "is_open": true,
        "custom_attrs": {
            "C-6702376000044400907": {

            "C-6702376000048595214": {
                "zh_cn": "自定义属性 2",
                "en_us": "custom_attr2",

            }
        }
    }
}
```
