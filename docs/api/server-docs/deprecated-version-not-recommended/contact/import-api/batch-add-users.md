# Batch add users

This API is used to add a batch of new users to Contacts.

> Permission to write to Contacts for the department to which the users belong is required to invoke this API.
> Marketplace Apps do not have permission to invoke this API.
> The application should apply for `update contacts` and `access contacts by application identity` permissions to invoke this API.
