# Report by department

It refers to data aggregated by department, which is the statistics of the users directly under the department, including the user data in the department dimension (the number of new and offboarded users, the number of new active users, etc.) and the function usage data of the department. The data of the day will be generated at 8:00 AM the next day (UTC+8).

## Methods

**Obtain active users and function usage data in the department dimension**

    The API is used to obtain active users and function usage data in the department dimension, including the usage data about IM (Instant Messaging), Calendar, Docs, and Audio/Video Conferencing.
