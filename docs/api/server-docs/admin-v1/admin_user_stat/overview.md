# Report by user

It refers to data aggregated by user, including data of this user (add/activation time, active device, etc.) and function usage data of this user. The data of the day will be generated at 5:00 am the next day (UTC+8).

## Methods

**Obtain active users and function usage data in the user dimension**

    This API is used to obtain active users and function usage data in the user dimension. That is, the usage data of IM (Instant messaging), Calendar, Docs, and audio/video conferencing functions.
