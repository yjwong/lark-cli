# Comment Features Overview

The comment interface can achieve functions such as creating, replying, updating, deleting, and getting details of comments.

When designing the comment interface, it can support commenting on any resource, but currently only supports task comments. In the future, it may support commenting on checklists, dynamics, and other resources.

Use the Create Comment interface to create a comment, or reply to an existing comment on the same task.

Use the Get Comment Details interface to obtain all data of a comment.

Use the Update Comment interface to modify the content of a comment. Only the creator of the comment can update it.

Use the Delete Comment interface to delete a comment. Only the creator of the comment can delete it. No action can be taken on a deleted comment. Deleted comments cannot be restored.

Use the Get Comment List interface to obtain the list of comments of a resource in a paginated form according to the creation time in ascending/descending order.
