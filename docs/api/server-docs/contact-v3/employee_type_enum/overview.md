# Overview
##  Resource definition
A special user attribute, used to mark the identity type of a user. Only one value can be selected for the field.

Default values include employee, intern, outsourcing, contractor, and consultant.
Companies can add custom values based on their actual conditions, and each user can be associated with only one value.

##  Field description
