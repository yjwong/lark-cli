# Resource introduction
"Role" refers to the concept of "role" in the "Role Management" module in the Lark management background. At present, it is mainly used in approval scenarios, which refers to the professional division of team members, which can be composed of one or more members, such as personnel, administration, finance, etc. The unique identifier of the role is "role_id", which is unique under the tenant.
