#  Overview
##  Resource definition
A node in the organizational structure tree of a company in Lark.

##  Field description
| Parameter | Type | Description |
| --- | --- | --- |
| `department_id` | `string` | Identifies a unique department within a tenant and supports custom or default ID generation. For details, see Department IDs. **Example value**: "h121921" **Data validation rules**: - Maximum length: `64` characters - Regular validation: `^0|[^od][A-Za-z0-9]*` |
| `open_department_id` | `string` | Identifies a department in an app, and a `department_id` value corresponds to the same `open_department_id` value in different apps. For details, see Department IDs. **Example value**: "od-4e6ac4d14bcd5071a37a39de902c7141" |
| `name` | `string` | Department name **Example value**: "DemoName" **Data validation rules**: - Minimum length: `1` character **Required field scopes (Satisfy any)**: Obtain department's basic information Read contacts as an app |
| `i18n_name` | `department_i18n_name` | Internationalized department name **Required field scopes (Satisfy any)**: Obtain department's basic information Read contacts as an app |

## Department IDs
### department_id

`department_id` is used to uniquely identify a department in a tenant. You can define it when creating a department or use the unique `department_id` automatically generated by the system.

The `department_id` of departments cannot be modified once created.

> **Note:** You can write existing unique IDs of your departments to department IDs in Lark to ensure consistency between Lark and your internal system, saving the mapping cost of cross-system calls.

### open_department_id
`open_department_id` is used to identify a department in an app. A `department_id` value corresponds to different `open_department_id` values in different apps.

### How to define department_id

When creating a department, write a custom department ID via the department_id parameter. The data validation rules are as follows:

- Maximum length: `64` characters
- Regular validation: `^0|[^od][A-Za-z0-9]*`

### How to obtain department_id

#### Suggestions
- If your department IDs are defined by yourself, we recommend that you maintain a list of department IDs locally for your convenience.
- If you do not maintain department IDs, you can ask the tenant administrator to search for a department ID in Tenant Admin.

#### How tenant administrators search for department IDs

Tenant administrators who can manage Member and Department can search for IDs of departments within their management scope.

- Method 1: Search for a single department ID

In the Admin > [Organization > Member and Department](http://www.larksuite.com/admin/contacts/departmentanduser), click "..." next to a department in the department tree on the left to open the editing drawer.
