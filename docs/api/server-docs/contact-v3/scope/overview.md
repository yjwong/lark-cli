# Overview
##  Resource definition
This resource refers to the scope of the organizational structure data that an app can access via the contacts API, which is defined by the [company administrator] (https://www.larksuite.com/hc/en-US/articles/360048488029) in [Admin > Workplace > App Management](http://www.larksuite.com/admin/appCenter/manage).

> **Note:** If you want to know whenever the company administrator changes the app's contact scope, subscribe to the Contact scope changed event, and then you'll be notified once any department/member is added to or deleted from the contact scope.
