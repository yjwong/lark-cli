#  List of country/region codes 
For Contacts user-related APIs, the user's country/region property (namely the "country" field) is saved and escaped as code data. If an invalid string is specified in the "country" field, the value may not be displayed in the Admin or the app. Therefore, always specify the field by following the code list below:
