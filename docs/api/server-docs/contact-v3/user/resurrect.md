# Restore deleted users

This interface is used to restore deleted users (resigned members).

## Usage restrictions

- This interface is only applicable to Lark Pro Edition and Lark Enterprise Edition. For version information, see [Version Comparison](https://www.larksuite.com/zh_cn/plans).
- This interface is only applicable to enterprise self-built applications. Store applications do not have permission to call this interface. For an introduction to application types, see Introduction to Application Types.

## Precautions
- Calling this interface only supports the restoration of members who have resigned within 30 days. After recovery, some user data is still unrecoverable, so please call with caution.
    - Recoverable data: individual chat records, external contacts, group chats, corporate email addresses and emails; untransferred documents, notes, and questionnaires.
    - Unrecoverable data: transferred resources, departments to which members belong, administrator roles, etc.
- The user ID of the member to be restored cannot be used by other members in the enterprise. If there are duplicates, please leave the member with the duplicate ID first, otherwise the interface will report an error.
- The mobile phone number and email address of the member to be restored cannot be used by other members in the company. If there are duplicates, please modify the mobile phone number or email address of the duplicate member first, otherwise the interface will report an error.
