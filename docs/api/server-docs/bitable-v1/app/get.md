# Get App Information

Get App information through app_token

## Request
| Facts |  |
| --- | --- |
| HTTP URL | https://open.larksuite.com/open-apis/bitable/v1/apps/:app_token |
| HTTP Method | GET |
| Rate Limit | 20 per second |
| Supported app types | custom,isv |
| Required scopes Enable any scope from the list | `base:app:read` `bitable:app` `bitable:app:readonly` | ### Request header
| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| Authorization | string | Yes | `tenant_access_token` or `user_access_token` **Value format**: "Bearer `access_token`" **Example value**: "Bearer u-7f1bcd13fc57d46bac21793a18e560" How to choose and get access token | ::: note
The instructions for AccessToken calling Docs API are detailed here Docs API Overview

### Path parameters
| Parameter | Type | Description |
| --- | --- | --- |
| `app_token` | `string` | Base unique device identifier app_token description **Example value**: "appbcbWCzen6D8dezhoCH2RpMAh" | ## Response

### Response body
| Parameter | Type | Description |
| --- | --- | --- |
| `code` | `int` | Error codes, fail if not zero |
| `msg` | `string` | Error descriptions |
| `data` | `\-` | \- |
|   `app` | `display_app` | App information |
|     `app_token` | `string` | AppToken information |
|     `name` | `string` | Name |
|     `revision` | `int` | Revison |
|     `is_advanced` | `boolean` | IsAdvanced |
|     `time_zone` | `string` | Base App Time Zone |
|     `formula_type` | `int` | Document formula field type Can be used in conjunction with field API. **Optional values are**:  - `1`: Formula fields do not support setting types - `2`: Formula fields support setting types  |
|     `advance_version` | `string` | Advanced version Can be used in conjunction with role API. **Optional values are**:  - `v1`: v1 - `v2`: v2  | ### Response body example

{
    "code": 0,
    "msg": "success",
    "data": {
        "app": {
            "app_token": "appbcbWCzen6D8dezhoCH2RpMAh",
            "name": "mybase",
            "revision": 1,
            "is_advanced": false,
            "time_zone": "Asia/Beijing",
            "formula_type": 1,
            "advance_version": "v1"
        }
    }
}
