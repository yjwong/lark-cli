# Streaming speech recognition (ASR)

A streaming speech recognition API is provided to input an audio clip by clip and receive recognition results in real time. Each audio clip is recommended to be within 100 to 200 ms.

> Single-tenant traffic limit: 20 channels .Single-tenant traffic limit: 20 QPS (each stream_id represents a session). Apps under the same tenant share the 20 QPS. The free version does not support calling this interface.

## Request
| Facts |  |
| --- | --- |
| HTTP URL | https://open.larksuite.com/open-apis/speech_to_text/v1/speech/stream_recognize |
| HTTP Method | POST |
| Supported app types | custom,isv |
| Required scopes | `speech_to_text:speech` | ### Request header
| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| Authorization | string | Yes | `tenant_access_token` **Value format**: "Bearer `access_token`" **Example value**: "Bearer t-7f1bcd13fc57d46bac21793a18e560" How to choose and get access token |
| Content-Type | string | Yes | **Fixed value**: "application/json; charset=utf-8" | ### Request body

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `speech` | `speech` | Yes | Audio resources |
|   `speech` | `string` | No | Content of the base64-encoded audio file **Example value**: "PdmrfE267Cd/Z9KpmNFh71A2PSJZxSp7+8upCg==" |
| `config` | `stream_config` | Yes | Configures properties |
|   `stream_id` | `string` | Yes | A 16-bit string generated by a user to identify the same data stream. The string can only contain letters, numbers, and underscores. **Example value**: "asd1234567890ddd" |
|   `sequence_id` | `int` | Yes | Sequence number of a data stream clip, starting from 0 and incremented by 1 for each request **Example value**: 1 |
|   `action` | `int` | Yes | Indicates the data packet sent to the API in a request. 1: The first packet. 2: The last packet. A result will be returned. 3: The cancelled data packets. No final result is returned. **Example value**: 1 |
|   `format` | `string` | Yes | Audio format. Only pcm supported. **Example value**: "pcm" |
|   `engine_type` | `string` | Yes | Engine type. Only 16k_auto that allows a mix of Chinese and English is supported. **Example value**: "16k_auto" | ### Request body example

{
    "speech": {
        "speech": "PdmrfE267Cd/Z9KpmNFh71A2PSJZxSp7+8upCg=="
    },
    "config": {
        "stream_id": "asd1234567890ddd",
        "sequence_id": 1,
        "action": 1,
        "format": "pcm",
        "engine_type": "16k_auto"
    }
}

## Response

### Response body
| Parameter | Type | Description |
| --- | --- | --- |
| `code` | `int` | Error codes, fail if not zero |
| `msg` | `string` | Error descriptions |
| `data` | `\-` | \- |
|   `stream_id` | `string` | A 16-bit random string that identifies the same data stream |
|   `sequence_id` | `int` | Sequence number of a data stream clip, starting from 0 and incremented by 1 for each request |
|   `recognition_text` | `string` | Text information after streaming speech recognition | ### Response body example

{
    "code": 0,
    "msg": "success",
    "data": {
        "stream_id": "asd1234567890ddd",
        "sequence_id": 1,
        "recognition_text": "Hello. You may want to try Lark."
    }
}

| 500 | 1610102 | network anomaly | Service or network error. Request again. |
