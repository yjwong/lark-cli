# Obtain attendance flow record

Obtains user attendance flow record via the attendance record ID.

## Request
| Facts |  |
| --- | --- |
| HTTP URL | https://open.larksuite.com/open-apis/attendance/v1/user_flows/:user_flow_id |
| HTTP Method | GET |
| Supported app types | custom |
| Required scopes Enable any scope from the list | `attendance:task` `attendance:task:readonly` | ### Request header
| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| Authorization | string | Yes | `tenant_access_token` **Value format**: "Bearer `access_token`" **Example value**: "Bearer t-7f1bcd13fc57d46bac21793a18e560" Learn more about obtaining and using access_token. | ### Path parameters
| Parameter | Type | Description |
| --- | --- | --- |
| `user_flow_id` | `string` | Attendance flow record ID, which is obtained as follows: 1) Batch query of attendance flow record 2) Obtain attendance results 3) Import attendance flow record **Example value**: "6708236686834352397" | ### Query parameters
| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `employee_type` | `string` | Yes | Type of employee ID for user_id and creator_id in response body **Example value**: "employee_id" **Optional values are**: - `open_id`: Open ID - `employee_id`: Employee's employee ID, that is the user ID in Lark Admin > Organization > Member and Department > Member Details - `employee_no`: Employee ID, that is the employee ID in Lark Admin > Organization > Member and Department > Member Details | ## Response

### Response body
| Parameter | Type | Description |
| --- | --- | --- |
| `code` | `int` | Error codes, fail if not zero |
| `msg` | `string` | Error descriptions |
| `data` | `user_flow` | \- |
| ∟ `user_id` | `string` | User ID |
| ∟ `creator_id` | `string` | Record creator ID |
| ∟ `location_name` | `string` | Attendance location name information |
| ∟ `check_time` | `string` | Attendance time, with the timestamp accurate to seconds |
| ∟ `comment` | `string` | Attendance note |
| ∟ `record_id` | `string` | Attendance record ID |
| ∟ `longitude` | `float` | Attendance longitude |
| ∟ `latitude` | `float` | Attendance latitude |
| ∟ `ssid` | `string` | SSID of attendance Wi-Fi |
| ∟ `bssid` | `string` | MAC address of attendance Wi-Fi |
| ∟ `is_field` | `boolean` | Whether it is offsite attendance |
| ∟ `is_wifi` | `boolean` | Whether Wi-Fi is used for attendance |
| ∟ `type` | `int` | Record generation method **Optional values are**: - `0`: User attendance - `1`: Administrator modification - `2`: User correction - `3`: Automatically generated by the system - `4`: Clock-out is not required - `5`: Attendance machine - `6`: Express attendance - `7`: Imports to attendance open platform |
| ∟ `photo_urls` | `string[]` | Attendance image list |
| ∟ `device_id` | `string` | Attendance device ID | ### Response body example
