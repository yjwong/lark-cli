# Mailing list resources

Mailing list resources

## Methods

**Create a mailing list**

    Creates a mailing list.

**Query the specified mail group**

    Obtains the information of a mailing list.

**Obtain mailing lists in batch**

    Obtains mailing lists by pages.

**Modify some information of mail group**

    Updates some fields of a mailing list. The fields not specified are not updated.

**Modify all information of mail group**

    Updates all information of a mailing list.

**Delete a mailing list**

    Deletes a mailing list.
