# Public mailbox resources

Public mailbox resources

## Methods

**Create a public mailbox**

    Creates a public mailbox.

**Query the specified public mailbox**

    Obtains public mailbox information.

**Check all public mailboxes**

    Obtains the list of public mailboxes by pages.

**Modify some information of public mailbox**

    Updates some fields of a public mailbox. The fields not specified are not updated.

**Modify all information of public mailbox**

    Updates all information of a public mailbox.

**Delete public mailbox address**

    Releases a public mailbox address from the email recycle bin. This deletes a public mailbox address forever, which means the mailbox address cannot be recovered once deleted.
