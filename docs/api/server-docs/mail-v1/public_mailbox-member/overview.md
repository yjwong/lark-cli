# Public mailbox member resources

Public mailbox member resources

## Methods

**Create a public mailbox member**

    Adds a member to a public mailbox.

**Get a public mailbox member**

    Obtains the information of a public mailbox member.

**List public mailbox members**

    Obtains the list of public mailbox members by pages.

**Delete a public mailbox member**

    Deletes a member of a public mailbox.

**Clear public mailbox members**

    Deletes all members of a public mailbox.
