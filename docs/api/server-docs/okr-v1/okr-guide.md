# OKR Development Guide 

## What is OKR

OKR（Objectives & Key Results）is a tool to achieve goal management and promote execution and cooperation:
- Objectives (O for short) are objectives, and give the answer for "what my team and I want to accomplish".
- Key Results (KR for short) are a series of measurable key results, and give the answer for "how I know if I have achieved my goal".
- KR is decomposed from O, and is used to support the realization of O. Ideally, when KR is complete, O is complete.

The process of writing OKR is also a process of goal definition, discussion, alignment and understanding, which will promote your in-depth thinking about the business.
